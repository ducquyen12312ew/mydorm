# Student Web App (Mobile-First)

This app is a dedicated student frontend built with React + Tailwind + Framer Motion.

## Run

```bash
npm install
npm run dev
```

Dev server: `http://localhost:5174`

It proxies API and socket traffic to backend `http://localhost:5000`.

## Build

```bash
npm run build
npm run preview
```

## Environment

Optional:
- `VITE_API_BASE_URL=/api/student-app`
- `VITE_SOCKET_URL=http://localhost:5000`
